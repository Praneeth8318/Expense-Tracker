# 💰 Expense Tracker

A simple and responsive Full Stack Expense Tracker built using **React**, **Node.js**, and **Express.js**. This application helps users manage their income and expenses while displaying the current balance and transaction history.

## 🚀 Features

- ➕ Add Income
- ➖ Add Expense
- 💰 Live Balance Calculation
- 📋 Transaction History
- 🗑️ Delete Transactions
- 📱 Clean and Responsive User Interface

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- JavaScript
- CSS

### Backend
- Node.js
- Express.js

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/Praneeth8318/Expense-Tracker.git
```

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm run dev
```
